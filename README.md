# 🖼️ Image Gallery

A responsive and interactive Image Gallery built using **HTML**, **CSS**, and **JavaScript**. This project demonstrates frontend web development skills by providing a clean and user-friendly interface for displaying images.

---

## 🚀 Features

- 📷 Responsive image gallery layout
- 🎨 Modern and clean user interface
- 📱 Mobile-friendly responsive design
- ⚡ Interactive image display
- 🖱️ Smooth user experience
- 💻 Built with pure HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
